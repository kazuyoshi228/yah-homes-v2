## 2. キーワード（フレーズ一致）

```
"福岡 住宿"
"福岡 民宿"
"博多 住宿"
"福岡 自由行 住宿"
"福岡 親子 住宿"
"九州 親子 住宿"
"福岡 整棟 住宿"
"福岡 airbnb"
```

- **「包棟」は使わない**（台湾語彙・前回の敗因仮説）。香港の語彙は 住宿・整棟・全屋。
- 「福岡 airbnb」は指名横取りではなく行動語彙（香港はAirbnb検索文化が強い）。CVが付きやすい想定。

## 3. 除外キーワード（キャンペーンレベル）

```
酒店 / 旅館 / 青年旅舍 / 膠囊 / 溫泉
熊本 / 大分 / 別府 / 由布院 / 鹿兒島 / 長崎 / 沖繩
機票 / 行程 / 攻略 / 天氣
koko premier / 利夫馬克斯 / service apartment
```

- 「酒店」（=ホテル。香港の宿泊検索の最大語彙だが業態ミスマッチ）を明示除外 — ここが台湾設定との最大の違い。
- 「九州 親子 住宿」が他県クエリを引くため九州の他都市名を先回りで遮断。
- 情報収集系（機票・行程・攻略・天氣）も遮断してクリックを購買意図に絞る。
