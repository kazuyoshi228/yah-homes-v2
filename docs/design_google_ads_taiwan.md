## 方針（日本キャンペーンとの違い）

- **主要KPIは「直販予約」ではなく「予約行動」**（台湾人はAirbnb/Agodaで最終予約する傾向。click_airbnb + click_booking_calendar の合算で評価）
- 広告文はAirbnb評価を前面に（遷移後の予約率が上がる）
- ランディングは /zh/（包棟語彙・FAQ 26問・親子住宿FAQ対応済み）
- **キャンペーンは日本と完全に分ける**（地域・言語・キーワード・広告文・LP すべて別）

## STEP 1: 新しいキャンペーン作成

Google Ads → 「+新しいキャンペーン」→「**ガイダンスなしでキャンペーンを作成**」→「**検索**」

コンバージョン目標: 「**予約**」1つだけ残す（他は削除）。掲載結果=「ウェブサイトへのアクセス」に `https://yah.homes/zh/`

## STEP 2: 単価設定

| 項目 | 値 |
|---|---|
| 重視している要素 | **クリック数**（データ0からのため。30件貯まったらコンバージョンへ） |
| 上限クリック単価 | **¥150**（台湾はCPCやや高めのため日本の¥100より上げる） |
| 新規ユーザー獲得を重視 | チェックしない |

## STEP 3: キャンペーン設定（最重要）

| 設定項目 | 値 | 注意 |
|---|---|---|
| Google 検索パートナー | **チェックを外す** | ★bot対策 |
| Google ディスプレイ ネットワーク | **チェックを外す** | ★bot対策 |
| 地域 | **台湾** | ラジオ「別の地域を入力する」→「台湾」を検索して追加 |
| 地域の一致 | **所在地**（「所在地やインタレスト」ではない） | 台湾在住者に限定 |
| 言語 | **中国語（繁体字）** | 「繁体字中国語」を追加 |
| AI最大化設定 | **オフ** | キーワード拡張を防ぐ |
| EU政治広告 | いいえ | |
| オーディエンス セグメント | 何も選ばない・モニタリングのまま | |

### 補足: 「日本にいる繁体字ユーザー」も狙うなら
実査で在日中国語話者の包棟検索が+∞成長。予算に余裕があれば、地域に「日本」も追加し、言語=繁体字で絞る別キャンペーンor広告グループを検討（訪日中に宿を探し直す層＝購買意欲最高）。まずは台湾単独でテスト。
